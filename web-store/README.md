# Rybella Web Store

متجر ويب متجاوب يعمل على الهاتف والحاسوب، مستوحى من التطبيق المحمول.

## التشغيل المحلي

```bash
npm install
npm run dev
```

يفتح على: **http://localhost:4001**

## التشغيل على السيرفر (البورت 4001)

```bash
# 1. ضبط عنوان الـ API (إن لزم)
# أنشئ ملف .env وأضف:
# VITE_API_URL=http://187.124.23.65:4000

# 2. البناء
npm run build

# 3. التشغيل
npm run start
```

المتجر يعمل على: **http://السيرفر:4001**

أو استخدم `npm run preview` بدلاً من `npm run start`.

## المميزات

- **الرئيسية:** Hero مع بحث، شبكة فئات، بانرات، منتجات شائعة ومميزة
- **استكشف:** فلترة حسب الفئة، الفئة الثانوية، العلامة، والمنتجات المميزة
- **الفئات والعلامات:** عرض جميع الفئات والعلامات
- **تفاصيل المنتج:** إضافة للسلة، اختيار اللون/الظل
- **السلة والدفع:** سلة ضيف + سلة مسجلة، إتمام الطلب
- **حسابي:** الملف الشخصي، الطلبات، المفضلة
- **التصميم:** مستوحى من BuyMore و child.com — شريط جانبي، بطاقات مستديرة، ألوان نعناعية

## المتطلبات

- Backend Rybella يعمل على `http://localhost:5000`
- أو ضبط `VITE_API_URL` في `.env` لرابط السيرفر
