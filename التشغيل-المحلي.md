# تشغيل لوحة التحكم محلياً

## الطريقة السريعة (موصى بها)

**انقر مرتين على:** `start-admin-local.bat`

سيقوم الملف بـ:
1. تثبيت المتطلبات
2. تشغيل Backend (المنفذ 5000)
3. تشغيل لوحة التحكم (المنفذ 3001)
4. فتح المتصفح تلقائياً

---

## التشغيل اليدوي

### 1. نافذة أولى - Backend
```cmd
cd backend
npm install
node server.js
```

### 2. نافذة ثانية - لوحة التحكم
```cmd
cd admin-dashboard
npm install
npm run dev
```

### 3. افتح المتصفح
http://localhost:3001

---

## الإعدادات المطبقة

- **تسجيل الدخول معطل** (SKIP_AUTH) - الدخول مباشرة بدون كلمة مرور
- **مسار API نسبي** - يتجنب مشاكل CORS
- **SQLite** - لا تحتاج MySQL

---

## إذا واجهت مشاكل

1. أغلق جميع نوافذ cmd المفتوحة
2. تأكد أن المنافذ 5000 و 3001 غير مستخدمة
3. شغّل الملف من جديد
