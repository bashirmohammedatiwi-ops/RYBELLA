# Rybella Iraq - متجر التجميل

المتجر الإلكتروني للعملاء - متصل بلوحة التحكم والـ Backend.

## التشغيل

1. تأكد من تشغيل **Backend** على المنفذ 5000
2. نفّذ من مجلد storefront:

```bash
npm install
npm run dev
```

3. افتح المتجر على: **http://localhost:3002**

## الصفحات

- **الرئيسية** `/` - Hero + الفئات + البانرات + المنتجات المميزة
- **المنتجات** `/products` - قائمة مع بحث وفلترة
- **تفاصيل المنتج** `/products/:id` - إضافة للسلة
- **الفئة** `/category/:id` - منتجات الفئة
- **السلة** `/cart` - عرض وتعديل السلة
- **إتمام الطلب** `/checkout` - يتطلب تسجيل الدخول
- **تسجيل الدخول** `/login`
- **إنشاء حساب** `/register`
- **طلباتي** `/orders` - يتطلب تسجيل الدخول

## التشغيل من الجذر

```bash
npm run store
```
