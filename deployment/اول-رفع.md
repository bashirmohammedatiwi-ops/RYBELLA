# أوامر الرفع لأول مرة على السيرفر

## الخطوات بالتسلسل

### 1. الاتصال بالسيرفر
```bash
ssh user@YOUR_SERVER_IP
```

---

### 2. تثبيت Docker و Docker Compose (إذا لم يكن مثبتاً)
```bash
sudo apt update
sudo apt install -y docker.io docker-compose-plugin
sudo systemctl enable docker
sudo systemctl start docker
sudo usermod -aG docker $USER
# اخرج وأعد الدخول لتطبيق usermod
```

---

### 3. استنساخ المشروع
```bash
git clone https://github.com/bashirmohammedatiwi-ops/RYBELLA.git
cd RYBELLA
```

---

### 4. إنشاء ملف البيئة
```bash
cp deployment/.env.example deployment/.env
```

---

### 5. تعديل ملف .env
```bash
nano deployment/.env
```
عدّل القيم:
- `JWT_SECRET` = مفتاح عشوائي قوي (مثال: `openssl rand -base64 32`)
- `API_URL` = رابط السيرفر (مثل `http://187.124.23.65:4000` أو `https://your-domain.com`)
- `HTTP_PORT` = 4000 (أو المنفذ المطلوب)

---

### 6. الدخول لمجلد deployment وبناء وتشغيل
```bash
cd deployment
docker compose up -d --build
```

---

### 7. التحقق من التشغيل
```bash
docker compose ps
docker compose logs -f
```
اضغط `Ctrl+C` للخروج من السجلات.

---

### 8. فتح لوحة التحكم
من المتصفح:
```
http://YOUR_SERVER_IP:4000
```

**تسجيل الدخول:**
- البريد: `admin@rybella.iq`
- كلمة المرور: `Admin@123`

---

## ملخص الأوامر (نسخ ولصق)

```bash
# 1. الاتصال
ssh user@YOUR_SERVER_IP

# 2. استنساخ
git clone https://github.com/bashirmohammedatiwi-ops/RYBELLA.git
cd RYBELLA

# 3. إنشاء .env
cp deployment/.env.example deployment/.env

# 4. تعديل .env (عدّل JWT_SECRET و API_URL)
nano deployment/.env

# 5. بناء وتشغيل
cd deployment
docker compose up -d --build

# 6. التحقق
docker compose ps
```
