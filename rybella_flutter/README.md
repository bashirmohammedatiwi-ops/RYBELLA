# Rybella Flutter - تطبيق موبايل

تطبيق Flutter يحتوي على جميع خصائص المتجر الإلكتروني Rybella بتصميم عالي الدقة وأداء ممتاز.

## المميزات

- **الرئيسية**: بانرات، عروض، فئات، منتجات مميزة، الأكثر مبيعاً، شاهدي مؤخراً
- **استكشف المنتجات**: فلترة بالفئات، العلامات التجارية، البحث، الترتيب
- **تفاصيل المنتج**: معرض صور، اختيار الدرجة/اللون، إضافة للسلة، المفضلة
- **السلة**: عرض العناصر، تعديل الكمية، حذف، إتمام الطلب
- **الطلبات**: عرض الطلبات وتفاصيلها
- **المفضلة**: قائمة المنتجات المفضلة
- **حسابي**: الطلبات، المفضلة، تسجيل الخروج
- **التسجيل والدخول**: إنشاء حساب، تسجيل الدخول

## التشغيل

```bash
# تثبيت الحزم
flutter pub get

# تشغيل على أندرويد
flutter run

# تشغيل مع عنوان API مخصص (للمحاكي)
flutter run --dart-define=API_BASE=http://10.0.2.2:4000
```

## إعدادات API

- **الإنتاج**: يستخدم `http://187.124.23.65:4000` افتراضياً
- **التطوير أندرويد محاكي**: `flutter run --dart-define=API_BASE=http://10.0.2.2:4000`
- **التطوير iOS محاكي**: `flutter run --dart-define=API_BASE=http://localhost:4000`

## البنية

```
lib/
├── core/           # الإعدادات، API، الثيم
├── models/         # نماذج البيانات
├── providers/      # Auth, Cart, Wishlist, RecentlyViewed
├── screens/        # الشاشات
├── services/       # ApiService
├── widgets/        # مكونات مشتركة
├── app_router.dart # التوجيه والتنقل
└── main.dart
```

## الحزم المستخدمة

- `provider` - إدارة الحالة
- `go_router` - التوجيه
- `cached_network_image` - صور مع تخزين مؤقت
- `http` - طلبات API
- `shared_preferences` - التخزين المحلي
- `google_fonts` - خط Tajawal
- `carousel_slider` - بانرات متحركة
